# zed-vala

[Vala](https://vala.dev) support for Zed.

# Features
- LSP (via [vala-language-server](https://github.com/vala-lang/vala-language-server))
- Formating via LSP using uncrustify
- Folding
- Syntax highlighting
- Outline
